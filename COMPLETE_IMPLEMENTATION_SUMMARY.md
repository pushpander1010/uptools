## Technical Implementation

### Files to Update Per Tool:
1. Create `/tool-name/index.html`
2. Add to `index.html` TOOLS array
3. Add to `wrangler.jsonc` watch_dir
4. Add to `public/sitemap.xml`

### Standard Tool Structure:
- Meta tags (title, description, OG, canonical)
- JSON-LD (SoftwareApplication, BreadcrumbList, FAQPage)
- 3-4 AdSense placements
- Input form with validation
- Results display with breakdown
- FAQ section (3+ questions)
- Breadcrumb navigation
- Mobile-responsive design

### Build & Deploy:
```bash
cd uptools
npm run build
wrangler deploy
```
