🎯 WhatsApp Tricks & Hidden Features

Unlock the full potential of WhatsApp with these pro tips and hidden features.

Messaging Tricks

📝 Text Formatting

*bold* for bold, _italic_ for italic, ~strikethrough~ for strikethrough, `monospace` for code

⏰ Schedule Messages

Long press send button → Select time → Message will be sent automatically at scheduled time

📌 Pin Important Messages

Long press message → Pin → Choose duration (8 hours, 24 hours, 7 days, or always)

🔍 Search Messages

Use search bar to find messages by keyword, date, or sender. Filter by media type.

Privacy & Security

👁️ Hide Last Seen

Settings → Privacy → Last Seen → Select "Nobody" to hide when you were last active

🔒 Lock Chats

Long press chat → Lock → Requires fingerprint/face ID to open sensitive conversations

🚫 Block & Report

Long press contact → Block → Blocked users can't see your status, last seen, or profile picture

Call Features

📞 Group Calls

Start a call → Tap add icon → Add up to 8 people for group video/audio calls

🎥 Screen Sharing

During video call → Tap share icon → Share your screen with the other person

Status & Profile

✨ Custom Status

Add text, emoji, or music to your status. Set expiration time (24 hours or custom)

🎨 Theme Customization

Settings → Chats → Theme → Choose Light, Dark, or System default

💡 Pro Tip: Use keyboard shortcuts on WhatsApp Web for faster messaging (Ctrl+N for new chat, Ctrl+Shift+M to mute).
