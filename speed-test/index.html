Frequently Asked Questions

How accurate is this speed test?
We use Cloudflare CDN edge servers (the same network behind speed.cloudflare.com) for download testing and our own Cloudflare Worker for upload. Each test runs 3 iterations and takes the median for consistency.
Does this store my data?
No. All measurements happen in your browser. We do not log, store, or transmit any speed test results to any server.
Why is my speed different from what my ISP promises?
ISP speeds are advertised as "up to" maximums. Actual speed depends on WiFi interference, distance from router, number of connected devices, server location, and network congestion. Run multiple tests at different times for the most accurate picture.
What units are used?
Speed is shown in Mbps (megabits per second). 1 Mbps = 0.125 MB/s. So 100 Mbps download = roughly 12.5 MB/s file download speed.
