📖 How to read a GSTIN

Digits 1–2
State Code (e.g. 27 = Maharashtra)
Digits 3–12
PAN Number of the taxpayer
Digit 13
Entity number (1–9, A–Z)
Digit 14
Always "Z" by default
Digit 15
Check digit (0–9 or A–Z)
