📐 How URL Shortener Works

1. Paste a long URL.
2. The tool calls the CleanURI API (free, no auth) to generate a short link.
3. A QR code is generated client-side using a pure JS QR library.
4. Links are saved in your browser history for easy access.

Are shortened links permanent?

Links via CleanURI have no expiration.

Can I create custom short links?

Yes, enter a custom alias in the optional field.

Is this tool free?

Yes, completely free. No signup required.
