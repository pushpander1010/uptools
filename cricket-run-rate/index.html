Frequently Asked Questions

How is run rate calculated in cricket?

Run rate (RR) = Total Runs Scored / Overs Bowled. For example, 160 runs in 20 overs = 8.00 run rate.

What is required run rate?

RRR tells how many runs per over are needed to reach the target. Formula: (Target - Current Runs) / Remaining Overs.

Does this work for Test cricket?
