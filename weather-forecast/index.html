📐 How Weather Forecast Works

1. Geocode: Your city name is converted to coordinates using OpenStreetMap Nominatim.
2. Fetch: Open-Meteo API returns current conditions + 7-day hourly data.
3. Display: Data is formatted with weather icons, temperatures, and wind details.

FAQ
Where does weather data come from?

Open-Meteo, a free open-source weather API powered by national weather services worldwide.

Is this tool free?

Yes, completely free with no API key required.

How accurate is the forecast?

Based on ECMWF and NOAA models. Accuracy decreases beyond 3-5 days.

Can I use it on mobile?

Yes, fully responsive design works great on all devices.
