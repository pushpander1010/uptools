📐 How Email Validation Works

1. Format Check: Validates against RFC 5322 regex pattern — checks for @, valid domain, proper characters.
2. Domain Check: Queries DNS-over-HTTPS to verify the domain exists and has MX (Mail Exchange) records.
3. Disposable Check: Cross-references against a database of 50+ known temporary/disposable email providers.
4. Scoring: Combines all checks into a trust score: Format (25%) + Domain (25%) + MX (25%) + Not Disposable (25%).

Frequently Asked Questions
How does email validation work?

It checks the format (regex), verifies the domain has MX records, and checks against disposable email providers.

Can I validate multiple emails at once?

Yes, switch to Batch mode to paste multiple emails and validate them all at once.

What is a disposable email?

Temporary addresses from services like Guerrilla Mail or TempMail, used to avoid giving out your real email.

Is this tool free?

Yes, completely free. No signup required. All validation happens in your browser.
